* Overview
:PROPERTIES:
  :CUSTOM_ID: introduction
  :END:

A modern, literate Emacs configuration using Org Mode's Babel format, emphasizing performance, language server integration, AI assistance, and productivity.

This project focuses on extensibility \emojifont{🚀 🧩 🤖 🎨 🗂️ ✅ 📦 🌿 📚 🧠} and maintainability.

** Design Tenets

- *Deterministic startup* — no implicit installs or hidden side effects.
- *Layered ownership* — each layer owns its policy; dependencies flow downward.
- *Opt-in power* — advanced tooling never required for baseline use.
- *Observable state* — prefer inspectable behavior over opaque automation.
- *Deletion-friendly* — removing a layer reduces convenience, not correctness.

** Configuration Philosophy

This configuration is not a collection of tweaks.
It is a deliberately layered system designed around
predictability, isolation, and long-term maintainability.

The guiding philosophy is summarized below.

*** 1. Determinism over Convenience

Startup must be reproducible.

- No implicit directory scanning
- No conditional auto-loading based on environment state
- No automatic installation of external components
- No hidden side effects during bootstrap

Every loaded module is explicitly declared.
Every dependency is intentional.

If something changes, it should be explainable.

*** 2. Isolation by Layer

Each layer owns a specific responsibility:

- early-init  → process-level invariants
- core        → global policy and infrastructure
- ui          → visual presentation
- completion  → minibuffer UX
- orgx        → Org extensions only
- dev         → development tooling
- utils       → narrow helper functions
- personal    → user- and device-specific overlays

Higher layers may depend on lower layers.
Lower layers must never depend on higher layers.

This directionality is strictly enforced.

*** 3. Explicit over Implicit

Implicit behavior is avoided whenever possible.

Examples:

- Tree-sitter grammars are never auto-installed.
- Native compilation warnings are intentionally configured.
- Module loading is fully enumerated.
- Global hooks are added only with documented intent.

Nothing activates itself silently.

*** 4. Safe Deletion

Any layer can be removed without breaking the system.

This implies:

- No circular dependencies
- No hidden cross-layer coupling
- No dynamic feature injection from outside the module graph

A removed module should fail cleanly.

*** 5. Observable State

State transitions must be inspectable.

- Module load timing can be reported.
- Failure to load a module is explicit.
- Directory creation failures emit warnings.
- Maintenance tasks are threshold-based and predictable.

Silence is intentional.
Noise is actionable.

*** 6. Forward Compatibility

The configuration anticipates future Emacs versions.

- Version guards protect compatibility fixes.
- Experimental features are opt-in.
- Native compilation and Tree-sitter are centrally controlled.
- Paths and caches follow XDG-like structure.

Upgrading Emacs should not require rewriting architecture.

*** 7. Policy vs. Preference

Core defines policy.
Personal defines preference.

Global invariants live in `core/`.
User taste and device specifics live in `personal/`.

This prevents accidental drift of shared behavior.

---

This philosophy ensures that the configuration
remains stable, debuggable, and maintainable
even as Emacs evolves.

* Configuration Files
:PROPERTIES:
:CUSTOM_ID: structure
:END:

This Emacs configuration is *modular by design* and targets **Emacs 30+**.
Each layer has a clearly defined responsibility to keep behavior predictable,
UI replaceable, and personal customizations isolated.

- =early-init.el= → earliest bootstrap (performance, paths, UI defaults)
- =init.el=       → package bootstrap, global defaults, module entrypoint
- =lisp/=         → shared, versioned modules (core, ui, completion, orgx, dev, vcs, utils)
- =personal/=     → user- and device-specific overlays (not shared policy)

** Core Bootstrap — early-init.el & init.el
:PROPERTIES:
:CUSTOM_ID: core-bootstrap
:END:

*** Overview

**** Purpose

Provide a clean, fast bootstrap that prepares Emacs before feature activation.

Initialization is split into two stages:

- =early-init.el= — pre-package bootstrap (directories, performance guards, early UI).
- =init.el= — package bootstrap (*straight.el + leaf*), runtime setup, and module entrypoint.

Infrastructure is established first; feature logic is activated later.

**** Responsibilities

This bootstrap layer:

- Disables =package.el=; uses *straight.el* exclusively.
- Optimizes startup (temporary GC/file-handler relaxation with safe restoration).
- Normalizes state under =.cache/=, =.etc/=, =.var/= (including native-comp artifacts).
- Prepares Homebrew toolchain variables on macOS before native compilation.
- Applies early UI defaults to avoid flicker.
- Configures URL state paths before =url.el= loads.
- Initializes *leaf* and conservative runtime helpers (GCMH, IO buffers).
- Provides two entrypoints:
  - =personal/<login-name>.el=
  - =lisp/modules.el=

**** Reproducibility

Shared layers are reproducible:

- =early-init.el=
- =init.el=
- =lisp/=

Personal overlays are intentionally user-scoped:

: personal/<login-name>.el

This favors isolation and multi-user safety over strict byte-for-byte reproducibility.

**** Module Map

| File            | Role |
|-----------------+--------------------------------------------|
| =early-init.el= | Pre-init infrastructure & performance guards |
| =init.el=       | Package bootstrap & module activation      |

**** Boot Flow

1. =early-init.el=
   - Establish directories
   - Disable =package.el=
   - Relax GC/file handlers
   - Apply early UI defaults
   - Prepare macOS toolchain (if applicable)

2. =init.el=
   - Configure URL state
   - Bootstrap *straight.el*
   - Import macOS login environment (GUI/daemon)
   - Initialize *leaf*
   - Apply runtime performance knobs
   - Load personal overlay
   - Require =modules.el=

3. After startup:
   - Print elapsed time and GC count.

**** Key Settings

- =package-enable-at-startup= :: =nil=
- =straight-base-dir= :: under =.cache/=
- =native-comp-eln-load-path= :: =.cache/eln-cache=
- =read-process-output-max= :: 4 MiB (temporary)
- =gcmh-high-cons-threshold= :: 16 MiB

**** Usage Guidelines

- Keep =early-init.el= infrastructure-only.
- Put shared behavior in modules via =modules.el=.
- Put user/device glue in =personal/=.
- Restart Emacs after toolchain upgrades.
- The configuration root is relocatable; state directories regenerate automatically.

**** Troubleshooting

- Native compilation fails on macOS →
  Ensure Homebrew =libgccjit= is installed and visible.

- Straight bootstrap fails →
  Usually a transient network issue; retry.

- Startup echo warning →
  =inhibit-startup-echo-area-message= must be a string.

*** early-init.el
:PROPERTIES:
:header-args:emacs-lisp: :tangle early-init.el
:END:

#+begin_src emacs-lisp
  ;;; early-init.el --- Early bootstrap and runtime foundations -*- lexical-binding: t; -*-
  ;;
  ;; Copyright (c) 2021-2026
  ;; Author: YAMASHITA, Takao
  ;; License: GNU GPL v3 or later
  ;;
  ;; Category: core
  ;;
  ;;; Commentary:
  ;; Early initialization executed before regular init.el.
  ;;
  ;; This file:
  ;; - disables package.el and quickstart
  ;; - performs startup optimizations (GC, file-name-handlers)
  ;; - defines base configuration directories (.var / .etc / .cache / lisp)
  ;; - configures native compilation and Tree-sitter paths
  ;; - prepares macOS Homebrew toolchain environment
  ;; - establishes early UI defaults and frame parameters
  ;;
  ;;; Code:

  (eval-when-compile
    (require 'subr-x))

  (require 'seq)

  ;; ---------------------------------------------------------------------------
  ;; Internal utilities
  ;; ---------------------------------------------------------------------------

  (defun core--ensure-directory (dir)
    "Ensure DIR exists, creating it recursively if needed."
    (unless (file-directory-p dir)
      (condition-case err
          (make-directory dir t)
        (error
         (warn "early-init: failed to create %s (%s)"
               dir (error-message-string err))))))

  (defun core--login-username ()
    "Return login username or nil."
    (ignore-errors (user-login-name)))

  (defalias 'my/ensure-directory-exists #'core--ensure-directory)

  ;; ---------------------------------------------------------------------------
  ;; 1) Disable package.el
  ;; ---------------------------------------------------------------------------

  (setq package-enable-at-startup nil
        package-quickstart nil)

  ;; ---------------------------------------------------------------------------
  ;; 2) Base directories (root, .var/.etc, lisp load-path)
  ;; ---------------------------------------------------------------------------

  (defvar my:d
    (file-name-as-directory
     (or (and load-file-name
              (file-name-directory (file-chase-links load-file-name)))
         user-emacs-directory))
    "Root directory of this Emacs configuration.")

  (setq user-emacs-directory my:d)

  (defconst my:d:var  (expand-file-name ".var/" my:d))
  (defconst my:d:etc  (expand-file-name ".etc/" my:d))
  (defconst my:d:lisp (expand-file-name "lisp/" my:d))

  (when (and (boundp 'my:d:lisp)
             (file-directory-p my:d:lisp))
    (dolist (dir (directory-files my:d:lisp t "\\`[^.]"))
      (when (file-directory-p dir)
        (add-to-list 'load-path dir))))

  ;; ---------------------------------------------------------------------------
  ;; 3) State directories (.cache, ELN, Tree-sitter, URL/EWW)
  ;; ---------------------------------------------------------------------------

  (defconst my:d:cache
    (expand-file-name
     "emacs/"
     (or (getenv "XDG_CACHE_HOME")
         (expand-file-name ".cache/" my:d))))
  (defconst my:d:eln-cache
    (expand-file-name "eln-cache/" my:d:cache))

  (defconst my:d:treesit
    (expand-file-name "tree-sitter/" my:d:var))

  (defconst my:d:url (expand-file-name "url/" my:d:var))
  (defconst my:d:eww (expand-file-name "eww/" my:d:var))

  (dolist (dir (list my:d:var my:d:etc my:d:lisp my:d:cache
                     my:d:eln-cache my:d:treesit my:d:url my:d:eww))
    (core--ensure-directory dir))

  (setq treesit-install-dir my:d:treesit)
  (setq treesit-extra-load-path (list my:d:treesit))

  ;; ---------------------------------------------------------------------------
  ;; 4) straight.el base
  ;; ---------------------------------------------------------------------------

  (setopt straight-base-dir my:d:cache
          straight-use-package-by-default t
          straight-vc-git-default-clone-depth 1
          straight-build-dir
          (format "build-%d.%d" emacs-major-version emacs-minor-version)
          straight-profiles '((nil . "default.el")))

  ;; ---------------------------------------------------------------------------
  ;; 5) macOS Homebrew toolchain
  ;; ---------------------------------------------------------------------------

  (when-let* ((brew (or (getenv "HOMEBREW_PREFIX")
                        (and (file-directory-p "/opt/homebrew") "/opt/homebrew")
                        (and (file-directory-p "/usr/local")   "/usr/local")))
              (bin  (expand-file-name "bin" brew)))
    (when (file-directory-p bin)
      (let* ((path  (or (getenv "PATH") ""))
             (parts (split-string path ":" t)))
        (unless (member bin parts)
          (setenv "PATH" (concat bin ":" path))))
      (let* ((gcc-candidates (directory-files bin t "^gcc-[0-9]+$"))
             (gcc-pairs
              (mapcar (lambda (path)
                        (when (string-match "gcc-\\([0-9]+\\)$" path)
                          (cons (string-to-number (match-string 1 path))
                                path)))
                      gcc-candidates))
             (gcc-sorted
              (sort (delq nil gcc-pairs)
                    (lambda (a b) (> (car a) (car b))))))
        (when-let* ((latest (cdr (car gcc-sorted))))
          (setenv "CC" latest)))))

  ;; ---------------------------------------------------------------------------
  ;; 6) Native compilation
  ;; ---------------------------------------------------------------------------

  (when (and (boundp 'native-comp-eln-load-path)
             (listp native-comp-eln-load-path))
    (setopt native-comp-eln-load-path
            (cons my:d:eln-cache
                  (delq my:d:eln-cache native-comp-eln-load-path))
            native-comp-async-report-warnings-errors 'silent))

  ;; ---------------------------------------------------------------------------
  ;; 7) no-littering compatibility
  ;; ---------------------------------------------------------------------------

  (defvar no-littering-etc-directory (file-name-as-directory my:d:etc))
  (defvar no-littering-var-directory (file-name-as-directory my:d:var))

  ;; ---------------------------------------------------------------------------
  ;; 8) Startup performance (GC + file-name-handler deferral)
  ;; ---------------------------------------------------------------------------

  (defvar core--orig-file-name-handler-alist file-name-handler-alist
    "Original `file-name-handler-alist' saved for restoration.")

  (defun core--restore-startup-state ()
    "Restore GC and file handler settings after startup."
    (setq file-name-handler-alist core--orig-file-name-handler-alist
          gc-cons-threshold 16777216
          gc-cons-percentage 0.1))

  (setq file-name-handler-alist nil
        gc-cons-threshold most-positive-fixnum
        gc-cons-percentage 0.6)

  (add-hook 'emacs-startup-hook #'core--restore-startup-state)

  ;; ---------------------------------------------------------------------------
  ;; 9) Backups / auto-save
  ;; ---------------------------------------------------------------------------

  (setq make-backup-files nil
        version-control nil
        delete-old-versions nil
        backup-by-copying nil
        auto-save-default nil
        auto-save-list-file-prefix nil)

  ;; ---------------------------------------------------------------------------
  ;; 10) Early UI defaults
  ;; ---------------------------------------------------------------------------

  (setopt frame-resize-pixelwise t
          frame-inhibit-implied-resize t
          cursor-in-non-selected-windows nil
          x-underline-at-descent-line t
          window-divider-default-right-width 16
          window-divider-default-places 'right-only)

  (dolist (it '((internal-border-width . 8)
                (tool-bar-lines . 0)))
    (add-to-list 'default-frame-alist it)
    (add-to-list 'initial-frame-alist it))

  (setq default-frame-alist
        (assq-delete-all 'fullscreen default-frame-alist))
  (setq initial-frame-alist
        (assq-delete-all 'fullscreen initial-frame-alist))

  (defvar core--did-fullscreen nil)

  (add-hook 'window-setup-hook
            (lambda ()
              (unless core--did-fullscreen
                (setq core--did-fullscreen t)
                (set-frame-parameter nil 'fullscreen 'fullboth))))

  (when (fboundp 'menu-bar-mode)   (menu-bar-mode -1))
  (when (fboundp 'tool-bar-mode)   (tool-bar-mode -1))
  (when (fboundp 'scroll-bar-mode) (scroll-bar-mode -1))

  ;; ---------------------------------------------------------------------------
  ;; 11) Startup echo
  ;; ---------------------------------------------------------------------------

  (when-let* ((u (core--login-username)))
    (setq inhibit-startup-echo-area-message u))

  (provide 'early-init)
  ;;; early-init.el ends here
#+end_src

*** init.el
:PROPERTIES:
:header-args:emacs-lisp: :tangle init.el
:END:

#+begin_src emacs-lisp
  ;;; init.el --- Main initialization entry point -*- lexical-binding: t; -*-
  ;;
  ;; Copyright (c) 2021-2026
  ;; Author: YAMASHITA, Takao
  ;; License: GNU GPL v3 or later
  ;;
  ;; Category: core
  ;;
  ;; Commentary:
  ;; Primary initialization sequence for Emacs 30+.
  ;;
  ;; This file:
  ;; - bootstraps straight.el, leaf, and Org
  ;; - establishes core performance and runtime defaults
  ;; - loads personal overlays safely
  ;; - invokes the modular configuration loader
  ;; - reports startup metrics after initialization
  ;;
  ;; Refactoring goals (no behavioral change):
  ;; - Clear phase separation (URL → bootstrap → runtime → personal → modules)
  ;; - Stronger logical grouping
  ;; - Constants for tunable runtime knobs
  ;; - Strict responsibility boundaries
  ;;
  ;;; Code:

  (require 'subr-x)
  (require 'seq)
  (require 'cl-lib)

  ;; ---------------------------------------------------------------------------
  ;; Internal helpers
  ;; ---------------------------------------------------------------------------

  (defun utils--safe-load-file (file &optional noerror)
    "Load FILE safely.
  If NOERROR is non-nil, log instead of raising."
    (when (and (stringp file) (file-exists-p file))
      (condition-case err
          (load file nil 'nomessage)
        (error
         (funcall (if noerror #'message #'user-error)
                  "[load] failed: %s (%s)"
                  file (error-message-string err))))))

  (defalias 'my/safe-load-file #'utils--safe-load-file)

  ;; ---------------------------------------------------------------------------
  ;; 0) URL state (must be set BEFORE requiring url.el)
  ;; ---------------------------------------------------------------------------

  (defvar core--url-state-dir
    (file-name-as-directory
     (or (bound-and-true-p my:d:url)
         (expand-file-name "url/" user-emacs-directory))))

  (setopt url-configuration-directory core--url-state-dir
          url-cookie-file (expand-file-name "cookies" core--url-state-dir)
          url-history-file (expand-file-name "history" core--url-state-dir)
          url-cache-directory (expand-file-name "cache/" core--url-state-dir))

  (dolist (d (list url-configuration-directory url-cache-directory))
    (make-directory d t))

  (require 'url)

  ;; ---------------------------------------------------------------------------
  ;; 1) Bootstrap straight.el (deterministic package management)
  ;; ---------------------------------------------------------------------------

  (defvar bootstrap-version 7)

  (let* ((base (or (bound-and-true-p straight-base-dir)
                   user-emacs-directory))
         (bootstrap-file
          (expand-file-name "straight/repos/straight.el/bootstrap.el" base)))
    (unless (file-exists-p bootstrap-file)
      (let ((buf
             (url-retrieve-synchronously
              "https://raw.githubusercontent.com/radian-software/straight.el/develop/install.el"
              'silent 'inhibit-cookies)))
        (unless (buffer-live-p buf)
          (user-error "[straight] failed to retrieve install.el"))
        (with-current-buffer buf
          (goto-char (point-max))
          (eval-print-last-sexp))))
    (load bootstrap-file nil 'nomessage))

  ;; ---------------------------------------------------------------------------
  ;; 2) leaf / org bootstrap (minimal, infrastructure-only)
  ;; ---------------------------------------------------------------------------

  (dolist (pkg '(leaf leaf-keywords))
    (straight-use-package pkg))

  (require 'leaf)

  (eval-when-compile
    (require 'leaf-keywords))

  (when (fboundp 'leaf-keywords-init)
    (leaf-keywords-init))

  (straight-use-package 'org)
  (require 'org)

  ;; ---------------------------------------------------------------------------
  ;; 3) macOS login environment import (GUI/daemon only)
  ;; ---------------------------------------------------------------------------

  (leaf exec-path-from-shell
    :straight t
    :when (and (eq system-type 'darwin)
               (or (daemonp) (memq window-system '(mac ns))))
    :config
    (setq exec-path-from-shell-check-startup-files nil
          exec-path-from-shell-arguments '("-l" "-i"))
    (exec-path-from-shell-copy-envs
     '("PATH" "LANG"
       "PASSWORD_STORE_DIR"
       "GPG_KEY_ID"
       "OPENROUTER_API_KEY"
       "OPENAI_API_KEY"))
    (exec-path-from-shell-initialize))

  ;; ---------------------------------------------------------------------------
  ;; 4) Runtime performance knobs (temporary expansion + safe restoration)
  ;; ---------------------------------------------------------------------------

  (defconst core-default-read-process-output-max
    (* 4 1024 1024)
    "Temporary large process output size during startup.")

  (defvar core--orig-read-process-output-max
    (and (boundp 'read-process-output-max)
         read-process-output-max))

  (when (boundp 'read-process-output-max)
    (setq read-process-output-max core-default-read-process-output-max))

  (add-hook 'after-init-hook
            (lambda ()
              (when (boundp 'read-process-output-max)
                (setq read-process-output-max
                      core--orig-read-process-output-max))))

  (leaf gcmh
    :straight t
    :custom
    ((gcmh-idle-delay . 5)
     (gcmh-high-cons-threshold . 16777216))
    :config
    (gcmh-mode 1))

  ;; ---------------------------------------------------------------------------
  ;; 5) Core built-ins (policy-level, no feature activation here)
  ;; ---------------------------------------------------------------------------

  (leaf emacs
    :straight nil
    :hook
    ((prog-mode . display-line-numbers-mode))
    :custom
    ((inhibit-startup-screen . t)
     (inhibit-startup-message . t)
     (initial-scratch-message . nil)
     (initial-major-mode . 'fundamental-mode)
     (use-short-answers . t)
     (create-lockfiles . nil)
     (idle-update-delay . 0.2)
     (ring-bell-function . #'ignore)
     (display-line-numbers-type . 'relative)
     (frame-title-format . t)
     (confirm-kill-emacs . #'y-or-n-p))
    :config
    (when (fboundp 'window-divider-mode)
      (window-divider-mode 1))
    (when (fboundp 'pixel-scroll-precision-mode)
      (pixel-scroll-precision-mode 1))
    (when (fboundp 'electric-pair-mode)
      (electric-pair-mode 1))
    (dolist (k '("C-z" "C-x C-z" "C-x C-c"))
      (keymap-global-unset k)))

  ;; ---------------------------------------------------------------------------
  ;; 6) Modifier key configuration (OS-specific, no side effects beyond vars)
  ;; ---------------------------------------------------------------------------

  (leaf my:modifier
    :straight nil
    :config
    (pcase system-type
      ('darwin
       (setq mac-option-modifier 'meta
             mac-command-modifier 'super
             mac-control-modifier 'control))
      ('windows-nt
       (setq w32-lwindow-modifier 'super
             w32-rwindow-modifier 'super))))

  ;; ---------------------------------------------------------------------------
  ;; 7) Personal overlay (strictly optional, safe-load only)
  ;; ---------------------------------------------------------------------------

  (let* ((root (or (bound-and-true-p my:d) user-emacs-directory))
         (personal (expand-file-name "personal/" root))
         (user (ignore-errors (user-login-name))))
    (when (file-directory-p personal)
      (add-to-list 'load-path personal))
    (my/safe-load-file (expand-file-name "user.el" personal) t)
    (when user
      (my/safe-load-file
       (expand-file-name (concat user ".el") personal) t)))

  ;; ---------------------------------------------------------------------------
  ;; 8) Modules entrypoint (authoritative feature activation)
  ;; ---------------------------------------------------------------------------

  (defvar my:modules-extra nil
    "Extra module list appended by optional layers.")

  (let* ((root (or (bound-and-true-p my:d) user-emacs-directory))
         (lisp-dir (expand-file-name "lisp/" root)))
    (when (file-directory-p lisp-dir)
      (add-to-list 'load-path lisp-dir))
    (require 'core-custom-ui-extras nil t)

** Modular Loader & Core Module Suite
:PROPERTIES:
:CUSTOM_ID: modular-loader-and-core-suite
:END:

*** Overview

This configuration uses a strictly layered, explicitly ordered
module loading architecture.

Shared behavior is activated only from a centralized entry point.
Modules are never discovered implicitly or loaded via directory scanning.

The goal is deterministic, reproducible, and inspectable startup.

**** Architecture

Initialization is segmented into three stages:

1. Infrastructure — =early-init.el=
2. Runtime setup   — =init.el=
3. Feature activation — =modules.el=

Only =modules.el= enables shared configuration modules.
Earlier stages prepare the environment but do not activate features.

**** Responsibility Model

Directories define bounded responsibility domains:

| Directory     | Responsibility |
|---------------+----------------|
| =core/=       | Runtime invariants and global policy |
| =ui/=         | Visual presentation and interaction |
| =completion/= | Completion frameworks |
| =orgx/=       | Org extensions |
| =dev/=        | Development tooling |
| =vcs/=        | Version control |
| =utils/=      | Narrow helpers |

Dependencies flow strictly downward.
Higher layers may depend on lower layers; the reverse is prohibited.

**** Authoritative Entry Point

=modules.el= defines:

- The ordered module list
- Error-isolated loading
- Skip and extension mechanisms
- Load timing diagnostics

It contains orchestration only.
Feature logic resides inside module files.

**** Load Order

1. =core=
2. =ui=
3. =completion=
4. =orgx=
5. =dev=
6. =vcs=
7. =utils=

This order encodes dependency direction.

**** Module Constraints

To preserve determinism:

- No directory scanning
- No wildcard expansion
- No environment-based conditionals
- No personal or host-specific logic

Each module must:

- Provide exactly one feature
- Be idempotent
- Avoid uncontrolled global side effects
- Remain safe under batch, byte-compilation, native-comp, and daemon contexts

**** Non-Responsibilities

The loader does not:

- Perform bootstrap or package initialization
- Establish runtime foundations
- Define hooks, advice, keybindings, or UI behavior
- Encode user- or host-specific policy

Its role is orchestration only.

**** Benefits

- Inspectable startup
- Predictable dependency flow
- Controlled diagnostics
- Stable compilation
- Safe layer removal (convenience lost, correctness preserved)

                      (float-time (time-subtract (current-time) m0)))))))

                   (length skipped)
                   (mapconcat #'symbol-name (nreverse skipped) " ")))
        (when failed
          (message "[modules] failed  (%d): %s"
                   (length failed)
                   (mapconcat #'symbol-name (nreverse failed) " "))))
      ok))

  ;;
  ;; Copyright (c) 2021-2026
  ;; Author: YAMASHITA, Takao
  ;; License: GNU GPL v3 or later
  ;;
  ;; Category: core
  ;;
  ;; Commentary:
  ;; Native compilation diagnostics and reporting policy for Emacs 30+.
  ;;
  ;; Goals:
  ;; - Keep default startup quiet (no noisy echoes).
  ;; - Optionally collect native-comp warnings for later inspection.
  ;; - Allow verbose mode for debugging/CI.
  ;; - Remain delete-safe: removing this module should not break correctness.
  ;;
  ;; This module:
  ;; - defines a 3-mode policy: silent / collect / verbose
  ;; - supports both async warning collection and explicit inspection commands
  ;; - integrates with CI via environment variables (optional and non-invasive)
  ;;
  ;; Notes:
  ;; - This module does not change native-comp directory layout. That belongs to
  ;;   early-init.el (eln-cache, load path).
  ;; - Collection is best-effort and should never abort startup.
  ;;
  ;;; Code:

  (eval-when-compile
    (require 'subr-x))

  (defgroup core-native nil
    "Native compilation diagnostics and reporting policy."
    :group 'convenience)

  (defcustom core-native-comp-report-mode 'collect
    "Native compilation reporting mode.

  - silent  : do not display or collect warnings (quiet baseline).
  - collect : display nothing, but collect warnings for later inspection.
  - verbose : display warnings (standard Emacs behavior).

  This module tries to implement these modes using the best available APIs
  for the running Emacs version (targeting Emacs 30+)."
    :type '(choice (const silent)
                   (const collect)
                   (const verbose))
    :group 'core-native)

  (defcustom core-native-comp-ci-force-verbose t
    "If non-nil, force verbose mode when running under CI.

  CI detection uses environment variable \"CI\" being set to a non-empty value."
    :type 'boolean
    :group 'core-native)

  (defcustom core-native-comp-collect-limit 2000
    "Maximum number of collected native-comp warning entries.

  When the limit is reached, oldest entries are dropped."
    :type 'integer
    :group 'core-native)

  (defcustom core-native-comp-warn-threshold 25
    "If collected warnings exceed this number, print a one-line notice.

  This never spams: it only emits once per session."
    :type 'integer
    :group 'core-native)

  (defcustom core-native-comp-auto-save-file nil
    "If non-nil, auto-save collected warnings to this file at Emacs exit.

  - nil: do not auto-save
  - string: path to a file (directories will be created)

  Saved format is plain text, append-safe is NOT used (the file is overwritten)."
    :type '(choice (const :tag "Disabled" nil)
                   (file  :tag "Path"))
    :group 'core-native)

  (defvar core-native--warnings nil
    "List of collected native-comp warning entries (newest first).")

  (defvar core-native--warned-threshold nil
    "Non-nil once the threshold notice has been emitted for this session.")

  (defvar core-native--installed-hook nil
    "Non-nil once we installed collection hooks for this session.")

  (defvar core-native--last-collect-at 0.0
    "Last time (float seconds) we collected warnings. Used for coarse throttling.")

  (defcustom core-native-comp-collect-min-interval 0.0
    "Minimum interval in seconds between successive collection runs.

  0 means no throttling. Non-zero can reduce overhead if Emacs emits many events."
    :type 'number
    :group 'core-native)

  (defun core-native--ci-p ()
    "Return non-nil if running under CI."
    (and core-native-comp-ci-force-verbose
         (let ((v (getenv "CI")))
           (and (stringp v) (not (string-empty-p v))))))

  (defun core-native--mode-effective ()
    "Return the effective report mode, considering CI forcing."
    (if (core-native--ci-p) 'verbose core-native-comp-report-mode))

  (defun core-native--cap-collect ()
    "Enforce `core-native-comp-collect-limit` on `core-native--warnings`."
    (when (and (integerp core-native-comp-collect-limit)
               (> core-native-comp-collect-limit 0))
      (when (> (length core-native--warnings) core-native-comp-collect-limit)
        (setcdr (nthcdr (1- core-native-comp-collect-limit) core-native--warnings) nil))))

  (defun core-native--maybe-notice-threshold ()
    "Emit a one-line notice if warning count exceeds threshold (once)."
    (when (and (not core-native--warned-threshold)
               (integerp core-native-comp-warn-threshold)
               (> core-native-comp-warn-threshold 0)
               (>= (length core-native--warnings) core-native-comp-warn-threshold))
      (setq core-native--warned-threshold t)
      (message "[core-native] collected %d native-comp warnings; use M-x core-native-show-warnings"
               (length core-native--warnings))))

  (defun core-native--normalize-entry (x)
    "Normalize X into a printable entry.

  We avoid strict assumptions about warning structure; we store what we got.
  Return a string."
    (cond
     ((stringp x) x)
     ((and (consp x) (stringp (car x)))
      ;; Some hooks may provide (MESSAGE . MORE) or (FILE . MESSAGE) etc.
      (format "%S" x))
     (t
      (format "%S" x))))

  (defun core-native--collect (warnings)
    "Collect native compilation WARNINGS (best-effort).

  WARNINGS may be:
  - list of strings
  - list of structured objects
  - a single string/object (depending on Emacs internals)

  We never throw; this must be safe in startup and hooks."
    (condition-case _err
        (let* ((now (float-time))
               (min-iv core-native-comp-collect-min-interval))
          (when (or (<= min-iv 0.0)
                    (>= (- now core-native--last-collect-at) min-iv))
            (setq core-native--last-collect-at now)
            (cond
             ((null warnings) nil)
             ((listp warnings)
              (dolist (w warnings)
                (push (core-native--normalize-entry w) core-native--warnings)))
             (t
              (push (core-native--normalize-entry warnings) core-native--warnings)))
            (core-native--cap-collect)
            (core-native--maybe-notice-threshold)))
      (error nil)))

  (defun core-native--install-collector ()
    "Install hooks/advice to collect async native-comp warnings when available."
    (unless core-native--installed-hook
      (setq core-native--installed-hook t)
      ;; Prefer official hook if present.
      (when (boundp 'comp-async-report-warnings-errors-hook)
        (add-hook 'comp-async-report-warnings-errors-hook #'core-native--collect))
      ;; Some builds may have alternative hooks; keep it conservative and optional.
      ;; If neither exists, collection mode becomes a no-op (delete-safe).
      ))

  (defun core-native--apply-report-setting (mode)
    "Apply MODE to native-comp reporting behavior."
    (pcase mode
      ('silent
       ;; Quiet: do not show.
       (when (boundp 'native-comp-async-report-warnings-errors)
         (setq native-comp-async-report-warnings-errors 'silent)))
      ('collect
       ;; Collect: we want warnings produced but not necessarily echoed.
       ;; Emacs uses `native-comp-async-report-warnings-errors` to control reporting.
       ;; Setting it to 'silent would suppress hook pathways in some cases, so we
       ;; keep it enabled (t) and ensure we collect via hook when available.
       (when (boundp 'native-comp-async-report-warnings-errors)
         (setq native-comp-async-report-warnings-errors t))
       (core-native--install-collector))
      ('verbose
       (when (boundp 'native-comp-async-report-warnings-errors)
         (setq native-comp-async-report-warnings-errors t)))
      (_
       ;; Unknown -> fall back to collect (safe, observable).
       (when (boundp 'native-comp-async-report-warnings-errors)
         (setq native-comp-async-report-warnings-errors t))
       (core-native--install-collector))))

  (defun core-native--save-to-file (path)
    "Save collected warnings to PATH (overwrite), creating directories as needed."
    (when (and (stringp path) (not (string-empty-p path)))
      (let ((dir (file-name-directory path)))
        (when (and dir (not (file-directory-p dir)))
          (make-directory dir t)))
      (with-temp-file path
        (insert ";; Native compilation warnings (collected)\n"
                ";; Generated by core/core-native.el\n\n")
        (if core-native--warnings
            (dolist (w (reverse core-native--warnings))
              (insert w "\n\n"))
          (insert "No warnings collected.\n")))))

  (defun core-native--maybe-auto-save ()
    "Auto-save collected warnings on exit if configured."
    (when (and core-native-comp-auto-save-file
               (stringp core-native-comp-auto-save-file))
      (condition-case err
          (core-native--save-to-file core-native-comp-auto-save-file)
        (error
         (message "[core-native] failed to save warnings: %s"
                  (error-message-string err))))))

  ;;;###autoload
  (defun core-native-clear-warnings ()
    "Clear collected native-comp warnings."
    (interactive)
    (setq core-native--warnings nil
          core-native--warned-threshold nil)
    (message "[core-native] cleared"))

  ;;;###autoload
  (defun core-native-warnings-count ()
    "Return the number of collected native-comp warning entries."
    (interactive)
    (message "[core-native] warnings=%d" (length core-native--warnings))
    (length core-native--warnings))

  ;;;###autoload
  (defun core-native-show-warnings ()

        (message "[core-native] no native-comp warnings recorded")
      (with-current-buffer (get-buffer-create "*native-comp-warnings*")
        (setq buffer-read-only nil)
        (erase-buffer)

                        (length core-native--warnings)))

          (insert w "\n\n"))

        (view-mode 1)
        (display-buffer (current-buffer)))))

  ;;;###autoload
  (defun core-native-save-warnings (path)
    "Save collected native-comp warnings to PATH (overwrite)."
    (interactive "FSave native-comp warnings to file: ")
    (core-native--save-to-file path)
    (message "[core-native] wrote %d warnings to %s"
             (length core-native--warnings)
             (expand-file-name path)))

  ;;;###autoload
  (defun core-native-set-mode (mode)
    "Set native-comp report MODE for this session.

  MODE is one of: silent, collect, verbose.

  This updates `core-native-comp-report-mode` and applies immediately."
    (interactive
     (list
      (intern
       (completing-read
        "core-native mode: "
        '("silent" "collect" "verbose")
        nil t nil nil (symbol-name (core-native--mode-effective))))))

             core-native-comp-report-mode
             (core-native--mode-effective)))

    "Apply native-comp diagnostics policy after init."
    (core-native--apply-report-setting (core-native--mode-effective))
    ;; Auto-save at exit (optional).

    "Default font size in points."
    :type 'integer
    :group 'ui-font)

  (defun ui-font--system-default ()
    "Return a default monospace font family depending on OS."
    (cond
     ((eq system-type 'darwin)     "Menlo")
     ((eq system-type 'gnu/linux)  "Monospace")
     ((eq system-type 'windows-nt) "Consolas")
     (t "Monospace")))

  (defun ui-font--system-emoji ()
    "Return a default emoji font family depending on OS."
    (cond
     ((eq system-type 'darwin)     "Apple Color Emoji")
     ((eq system-type 'gnu/linux)  "Noto Color Emoji")
     ((eq system-type 'windows-nt) "Segoe UI Emoji")
     (t "Noto Color Emoji")))

  (defun ui-font-apply ()
    "Apply font settings to the current frame."
    (when (display-graphic-p)
      (set-face-attribute
       'default nil
       :family (or ui-font-default (ui-font--system-default))
       :height (* 10 ui-font-size))
      (set-face-attribute
       'variable-pitch nil
       :family (or ui-font-variable-pitch
                   ui-font-default
                   (ui-font--system-default)))
      (set-fontset-font
       t 'emoji
       (font-spec :family

      (ui-font-describe)))

           (default-height (face-attribute 'default :height))
           (variable-family (face-attribute 'variable-pitch :family))
           (emoji-font (font-get
                        (font-spec :script 'emoji)
                        :family)))
      (message
       (concat
        "Fonts applied:\n"
        "  Default        : %s (%.1f pt)\n"
        "  Variable-pitch : %s\n"
        "  Emoji          : %s")
       default-family
       (/ default-height 10.0)
       variable-family
       (or emoji-font "unknown"))))

    :hook (prog-mode-hook . global-ligature-mode)

  (provide 'ui-font)

  ;;
  ;; Copyright (c) 2021-2026

  ;; License: GNU GPL v3 or later
  ;;

  ;;

  ;; - shuts down stale LSP servers safely
  ;; - integrates with project.el hooks only
  ;;
  ;;; Code:

  (leaf nil
    :straight nil
    :init
    (defcustom completion-lsp-enable-p t
      "Enable LSP lifecycle cleanup."
      :type 'boolean
      :group 'completion)

    (defvar completion-lsp--current-project-root nil)

    (defun completion-lsp--project-root ()
      (when-let* ((project (project-current nil))
  		(roots (project-roots project)))
        (car roots)))

    (defun completion-lsp-on-project-switch ()
      "Shutdown obsolete eglot servers on project switch."
      (when completion-lsp-enable-p
        (let ((new-root (completion-lsp--project-root)))
          (when (and completion-lsp--current-project-root
                     new-root
                     (not (string-equal completion-lsp--current-project-root new-root))
                     (featurep 'eglot))
            (dolist (server eglot--servers)
              (when (string-prefix-p completion-lsp--current-project-root
                                     (eglot--project-root (cdr server)))
                (ignore-errors
                  (eglot-shutdown (cdr server))))))
          (setq completion-lsp--current-project-root new-root))))

    (add-hook 'find-file-hook #'completion-lsp-on-project-switch)
    (add-hook 'project-switch-project-hook #'completion-lsp-on-project-switch))

*** Overview

**** Personal Layer Philosophy

The =personal/= layer provides user- and device-specific overlays
on top of the shared configuration.

It encodes identity, environment, and workflow glue
without altering global policy or architectural boundaries.

Its purpose is to keep shared layers:

- reproducible
- auditable
- portable

while allowing the system to feel native on a specific device.

**** Scope

This layer MAY contain:

- Identity (name, email)
- Personal feature switches (UI, LSP, thresholds)
- OS/device glue (IME, scrolling, platform quirks)
- Personal keybindings
- Optional integrations with user-owned services

This layer MUST NOT contain:

- Cross-user policy
- Shared defaults
- Modules depended on by other layers
- Startup-critical logic

Hooks/timers are allowed only if:

- device-local
- defensive (safe to no-op)
- non-invasive

**** Responsibilities

Concretely, this layer:

- Defines identity and safety preferences
- Selects UI/LSP stacks (=my:use-ui=, =my:use-lsp=)
- Binds shared infrastructure to local filesystem layout
- Wires Org directories and agenda scope
- Applies macOS-specific input and cursor behavior
- Tunes device-specific mouse/scroll behavior
- Exposes optional Apple Music controls (macOS)

**** Module Map

| Module file | Role |
|-------------+--------------------------------|
| =personal/user.el=          | Identity, fonts, UI/LSP switches, Org paths |
| =personal/device-darwin.el= | macOS device & IME glue |
| =personal/apple-music.el=   | Apple Music control (AppleScript + Hydra) |

**** Execution Model

1. Establish identity, fonts, and stack switches
2. Configure Org directories and agenda scope
3. Apply hygiene (load-path filtering)
4. Install macOS glue (IME, cursor sync)
5. Apply device profile (scroll/mouse tuning)
6. Register optional Apple Music commands

All macOS integrations are guarded via =fboundp= checks.

**** Key Settings

- =my:font-default= :: "JetBrains Mono"
- =my:font-variable-pitch= :: "Noto Sans JP"

  - =scroll-preserve-screen-position=t=

**** Usage

- Switch UI/LSP by changing =my:use-ui= / =my:use-lsp=
- Adjust fonts via personal variables
- Control Apple Music with =C-c M=
- Exclude agenda files via =archives/= or filename pattern

**** Troubleshooting

- IME switching fails →
  Verify macOS input source IDs.

Purpose:
Control Apple Music from Emacs on macOS.

What it does:
- Provides async/sync AppleScript helpers
- Defines interactive playback commands
- Exposes a hydra and optional meow leader binding

Notes:

#+begin_src emacs-lisp
  ;;; personal/apple-music.el --- Apple Music integration -*- lexical-binding: t; -*-
  ;;
  ;; Copyright (c) 2021-2026
  ;; Author: YAMASHITA, Takao
  ;; License: GNU GPL v3 or later
  ;;
  ;; Category: personal
  ;;
  ;; Commentary:
  ;; Control Apple Music using AppleScript.
  ;;
  ;; This module:
  ;; - provides asynchronous and synchronous AppleScript helpers
  ;; - defines interactive commands to control Apple Music playback
  ;; - integrates playback control with hydra and modal key systems
  ;;
  ;;; Code:

  (when (eq system-type 'darwin)

    ;; -------------------------------------------------------------------------
    ;; AppleScript helpers
    ;; -------------------------------------------------------------------------

    (defun my/apple-music--osascript-async (script &optional callback)
      "Run AppleScript SCRIPT asynchronously.
  If CALLBACK is non-nil, call it with the trimmed output."
      (let* ((proc-name "apple-music-async")
             (buffer-name "*Apple Music Async*")
             (proc (apply #'start-process
                          proc-name buffer-name
                          (list "osascript" "-e" script))))
        (when callback
          (set-process-sentinel
           proc
           (lambda (process event)
             (when (string= event "finished\n")
               (with-current-buffer (process-buffer process)
                 (funcall callback
                          (string-trim (buffer-string))))
               (kill-buffer (process-buffer process))))))))

    (defun my/apple-music--osascript-sync (script)
    "Run AppleScript SCRIPT synchronously and return trimmed output."
    (with-temp-buffer
      (let ((exit (process-file "osascript" nil t nil "-e" script)))
        (if (eq exit 0)
            (string-trim (buffer-string))
          (error "osascript failed: %s" (string-trim (buffer-string)))))))

    ;; -------------------------------------------------------------------------
    ;; Interactive commands
    ;; -------------------------------------------------------------------------

    ;;;###autoload
    (defun my/apple-music-play-pause ()
      "Toggle play/pause in Apple Music."
      (interactive)
      (my/apple-music--osascript-async
       "tell application \"Music\" to playpause"))

    ;;;###autoload
    (defun my/apple-music-next-track ()
      "Skip to the next track in Apple Music."
      (interactive)
      (my/apple-music--osascript-async
       "tell application \"Music\" to next track"))

    ;;;###autoload
    (defun my/apple-music-previous-track ()
      "Return to the previous track in Apple Music."
      (interactive)
      (my/apple-music--osascript-async
       "tell application \"Music\" to previous track"))

    ;;;###autoload
    (defun my/apple-music-current-track-info ()
      "Display current track information."
      (interactive)
      (message "%s"
               (my/apple-music--osascript-sync
                "tell application \"Music\" \
  to (get name of current track) & \" — \" & (get artist of current track) \
  & \" [\" & (get album of current track) & \"]\"")))

    (defun my/apple-music-get-playlists ()
      "Return a list of playlist names."
      (split-string
       (my/apple-music--osascript-sync
        "tell application \"Music\" to get name of playlists")
       ", "))

    ;;;###autoload
    (defun my/apple-music-play-playlist (playlist)
      "Prompt for PLAYLIST and play it."
      (interactive
       (list (completing-read
              "Playlist: "
              (my/apple-music-get-playlists))))
      (my/apple-music--osascript-async
       (format "tell application \"Music\" to play playlist \"%s\""
  	     (replace-regexp-in-string "\"" "\\\\\"" playlist))))

    ;; Hydra / meow integration
    ;; -------------------------------------------------------------------------

    (with-eval-after-load 'hydra
      (defhydra my/hydra-apple-music (:hint nil)
        "
  Apple Music
  -----------
  _p_: Play/Pause   _n_: Next   _b_: Back
  _l_: Playlist     _i_: Info   _q_: Quit

        ("p" my/apple-music-play-pause)
        ("n" my/apple-music-next-track)
        ("b" my/apple-music-previous-track)
        ("l" my/apple-music-play-playlist)
        ("i" my/apple-music-current-track-info)
        ("q" nil "quit"))
      (global-set-key (kbd "C-c M") #'my/hydra-apple-music/body))

    (with-eval-after-load 'meow
      (with-eval-after-load 'hydra
        (when (fboundp 'meow-leader-define-key)
          (meow-leader-define-key
           '("M" . my/hydra-apple-music/body))))))

#+begin_src emacs-lisp
  ;;; personal/personal-messages.el --- Messages archive integration -*- lexical-binding: t; -*-
  ;;
  ;; Category: personal
  ;;

  (eval-when-compile (require 'leaf))

  (require 'org)
  (require 'subr-x)

  (defgroup personal-messages nil
    "macOS Messages archiver."
    :group 'personal)

  (defcustom personal-messages-db
    (expand-file-name "~/Library/Messages/chat.db")
    "Path to macOS Messages database."
    :type 'string)

  (defcustom personal-messages-dir
    (expand-file-name "~/org/messages/")
    "Root directory for archived messages."
    :type 'string)

  (defcustom personal-messages-last-ts-file
    (expand-file-name "~/.emacs.d/messages-last-ts")
    "File storing last archived unix timestamp."
    :type 'string)

  (defun personal-messages--ensure-dir (dir)
    (unless (file-directory-p dir)
      (make-directory dir t)))

  (defun personal-messages--sanitize (name)
    (replace-regexp-in-string "[/:*?\"<>|]" "_" name))

  (defun personal-messages--ns-to-unix (ns)
    "Convert macOS nanosecond timestamp (since 2001-01-01) to Unix time."
    (+ (/ ns 1000000000) 978307200))

  (defun personal-messages--get-last-ts ()
    (if (file-exists-p personal-messages-last-ts-file)
        (string-to-number
         (string-trim
          (with-temp-buffer
            (insert-file-contents personal-messages-last-ts-file)
            (buffer-string))))
      0))

  (defun personal-messages--set-last-ts (ts)
    (with-temp-file personal-messages-last-ts-file
      (insert (number-to-string ts))))

  (defun personal-messages--target-file (handle ts)
    (let* ((time (seconds-to-time ts))
           (year (format-time-string "%Y" time))
           (month (format-time-string "%Y-%m" time))
           (base (expand-file-name
                  (personal-messages--sanitize handle)
                  personal-messages-dir))
           (year-dir (expand-file-name year base)))
      (personal-messages--ensure-dir year-dir)
      (expand-file-name (concat month ".org") year-dir)))

  (defun personal-messages--ensure-roam-header (file handle)
    (with-current-buffer (find-file-noselect file)
      (goto-char (point-min))
      (unless (re-search-forward "^#\\+title:" nil t)
        (insert
         (format "#+title: Conversation with %s\n#+filetags: :message:person:\n\n"
                 handle))
        (save-buffer))))

  (defun personal-messages--fetch-new ()
    (let* ((last-ts (personal-messages--get-last-ts))
           (sep "|")
           (sql (format
                 "SELECT h.id || '|' || m.date || '|' || m.is_from_me || '|' || replace(m.text, char(10), ' ') FROM message m LEFT JOIN handle h ON m.handle_id = h.ROWID WHERE m.text IS NOT NULL AND m.date > %d ORDER BY h.id, m.date ASC;"
                 last-ts))
           raw)

      (with-temp-buffer
        (call-process "sqlite3" nil t nil
                      "-batch"
                      "-noheader"
                      personal-messages-db
                      sql)
        (setq raw (buffer-string)))

      (let ((lines (split-string raw "\n" t))
            results)
        (dolist (line lines)
          (let ((parts (split-string line sep)))
            (when (= (length parts) 4)
              (push
               (list
                (nth 0 parts)
                (string-to-number (nth 1 parts)) ;; raw date
                (string-to-number (nth 2 parts))
                (nth 3 parts))
               results))))
        (nreverse results))))

  (defun personal-messages--write-entry (handle ts is-from-me text)
    (let* ((unix-ts (personal-messages--ns-to-unix ts))
           (file (personal-messages--target-file handle unix-ts))
           (time-str (format-time-string "%Y-%m-%d %H:%M"
                                         (seconds-to-time unix-ts)))
           (from (if (= is-from-me 1) "me" handle)))

      (personal-messages--ensure-roam-header file handle)

      (with-current-buffer (find-file-noselect file)
        (goto-char (point-max))
        (insert
         (format "* %s\n:PROPERTIES:\n:RAW_NS: %d\n:FROM: %s\n:END:\n%s\n\n"
                 time-str ts from text))
        (save-buffer))))

  (defun personal-messages-archive ()
    "Archive new messages by handle."
    (interactive)
    (personal-messages--ensure-dir personal-messages-dir)
    (let ((msgs (personal-messages--fetch-new))
          (max-ts 0))
      (dolist (m msgs)
        (let ((handle (nth 0 m))
              (ts (nth 1 m))
              (is-from-me (nth 2 m))
              (text (nth 3 m)))
          (personal-messages--write-entry
           handle ts is-from-me text)
          (setq max-ts (max max-ts ts))))
      (when (> max-ts 0)
        (personal-messages--set-last-ts max-ts))))

  (provide 'personal-messages)
  ;;; personal/personal-messages.el ends here
#+end_src
